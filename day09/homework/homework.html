JavaScript

JavaScript (JS) არის პროგრამირების ენა, რომელიც ჩვეულებრივ გამოიყენება დინამიური და ინტერაქტიული ელემენტების შესაქმნელად ვებსაიტებზე. ის ერთ-ერთი ყველაზე პოპულარული ენაა და მნიშვნელოვან როლს ასრულებს მოდერნული ვებ დეველოპმენტში. JavaScript-ს იყენებენ როგორც კლიენტ-თანხმად (ბრაუზერში შესრულებისთვის), ისე სერვერზე (Node.js).

ისტორია

JavaScript შეიქმნა 1995 წელს ბრენდან აიიხმა (Brendan Eich) Netscape Communications Corp.-ში. თავდაპირველად ენა ჩაიწერა როგორც LiveScript, მაგრამ მოგვიანებით, როგორც JavaScript, რათა დაერწმუნებინათ მომხმარებლები მისი კავშირი ჰქონოდა Java-სთან. თუმცა, JavaScript და Java არა მხოლოდ სახელწოდების, არამედ ფუნქციონალობის თვალსაზრისითაც განსხვავდებიან.

მახასიათებლები

JavaScript მუშაობს როგორც ინტერაქტიული ენა, რომელიც მომხმარებლის ქმედებებზე რეაგირებს და შეიძლება გამოიყენებოდეს სხვადასხვა მახასიათებლებისთვის: დოკუმენტების მოდიფიკაცია - HTML და CSS-ის მეშვეობით, JavaScript ქმნის დინამიურ ვებსაიტებს. მომხმარებლის ინპუტის დამუშავება - ფორუმებზე, ძიების მინდვრებზე ან სხვადასხვა ფორმებზე. დასაქმებული ფაილების ან სხვა რესურსების დამუშავება - JavaScript-ს შეუძლია მონაცემების ამოღება და გადამუშავება. ტრანზაქციების და არა ბლოკირებადი ოპერაციების შესრულება - ასინქრონული კოდით, მაგალითად, AJAX-ის მეშვეობით.
